# Pyramid
Write a function that accepts a single parameter, an Integer that creates an octothorpe (`#`) pyramid. The parameter determines the width (in octothorpes) of the bottom tier of the pyramid.

#### Example input 1
```ruby
pyramid(7)
```

#### Expected output 1
```
   #
  ###
 #####
#######
```

#### Example input 2
```ruby
pyramid(6)
```

#### Expected output 2
```
  ##
 ####
######
```
